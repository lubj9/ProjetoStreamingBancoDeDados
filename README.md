# 🎬 Projeto Streaming Database

## 📌 Sobre o Projeto

O Projeto Streaming Database é um sistema de banco de dados relacional desenvolvido para simular o funcionamento de uma plataforma de streaming digital. O objetivo é gerenciar usuários, conteúdos, assinaturas, histórico de visualizações e avaliações, aplicando conceitos fundamentais de modelagem e implementação de bancos de dados.

Este projeto foi desenvolvido como parte dos estudos em Banco de Dados no curso de Sistemas de Informação, visando consolidar conhecimentos em modelagem conceitual, normalização, integridade referencial e consultas SQL.

---

## 🚀 Funcionalidades

* Cadastro de usuários
* Gerenciamento de conteúdos (filmes, séries e documentários)
* Controle de assinaturas
* Registro de histórico de visualizações
* Sistema de avaliações dos conteúdos
* Consultas analíticas utilizando SQL
* Relacionamentos entre entidades com integridade referencial

---

## 🏗️ Modelagem do Banco

O banco foi estruturado seguindo as etapas:

1. Modelagem Conceitual (MER)
2. Modelagem Lógica
3. Modelo Relacional
4. Implementação Física em SQL

Foram aplicados conceitos de:

* Chaves Primárias (PK)
* Chaves Estrangeiras (FK)
* Relacionamentos 1:N e N:N
* Normalização
* Integridade Referencial
